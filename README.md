# Sistema Bancário Simples em Java

Trabalho desenvolvido para a disciplina **Boas Práticas de Programação**.  
O objetivo foi implementar um sistema bancário simples usando **orientação a objetos** e aplicando **boas práticas**.

## Descrição do Projeto

O sistema simula operações básicas de um banco:

- Cadastro de **clientes**
- Criação de **contas correntes** e **contas poupança**
- **Depósito**, **saque** e **transferência** entre contas
- **Consulta de saldo** de uma conta específica
- Aplicação de **rendimentos** nas contas poupança
- **Listagem** de todas as contas por ordem de saldo
- **Geração de relatório** de consolidação do banco

Toda a interação é feita por meio de um **menu textual** exibido no terminal, ao executar o programa principal.

---

## Para Executar:
Tenha instalado o JDK no seu computador

Clone o repositório, e entre na pasta

Para compilar os arquivos use o comando:
```javac *.java```

Para executar o programa, depois de compilar, use o comando:
```java Main```
